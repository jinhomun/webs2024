01. Math.random() : 랜덤값 반환(0~1 사이의 난수 값을 반환) : 반환(숫자)

Math.random() 함수는 0 이상 1 미만의 구간에서 근사적으로 균일한(approximately uniform) 부동소숫점 의사난수를 반환하며, 이 값은 사용자가 원하는 범위로 변형할 수 있다. 

{
    const x = Math.random();
    console.log(x);
}
