  • 01. 이벤트 메서드 : addEventListener("mousedown") : 마우스 버튼을 클릭했을 때

  • 06. 이벤트 메서드 : addEventListener("mouseover") : 요소 위에 포인터 요소 위치가 있을 때
  • 07. 이벤트 메서드 : addEventListener("mouseout") : 요소 위에 포인터 요소 위치가 벗어 났을 때
